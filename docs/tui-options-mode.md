# `/options` Mode

Settings and configuration. At minimum:

- **Default curriculum** — set a default curriculum so `/learn` doesn't always prompt for one.
- **Agent model / provider** — configure which LLM backend to use.
- **Commit defaults** — default `entry_type`, auto-tagging behavior, etc.
- **Review settings** — default quiz length, timer duration for timed quizzes.
- **Database management** — export/import, reset, view stats (entry counts, tag usage, etc.).
- **Integrations (future)** — Anki export, Obsidian export.
