### Core Ideas
Some way to create and manage "knowledge bases" (curriculums) through an agent.
* "Learning" phase - ask questions about how to do things, automatically add those as examples to a database of knowledge on the subject
* "Practice" phase - generate quizzes on certain knowledge on the fly
	* For certain topics, generate "timed quizzes" to practice things you want to be readily at hand, and track performance over time (e.g., "vim motions", CLI tools, etc.)
* Anki integration - automatically export to flash cards
* Obsidian integration (maybe export explanations to notes)
